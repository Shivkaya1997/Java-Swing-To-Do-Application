# Java-Swing-To-Do-Application
To create a functional user interface that allows users to manage daily tasks through a graphical window


Features
Add Tasks: Input text and add it to the list.
Delete Tasks: Select a task from the list and remove it.
Keyboard Support: Use the Enter key to add tasks quickly.
Scrollable List: Handles long lists of tasks using a scroll pane.

Tools and Technologies
Language: Java
Framework: Swing
Editor: VS Code


Terminal: Command line for compilation and execution
Project Structure
TodoApp.java: Main source code file.
README.md: Project documentation.


How to Run
Save the code as TodoApp.java.
Open your terminal.
Compile the code using: javac TodoApp.java
Run the application using: java TodoApp
Usage Instructions
Type a task in the input box at the top.
Click the Add Task button.
To remove an item, click the task in the list and click Delete Selected.
